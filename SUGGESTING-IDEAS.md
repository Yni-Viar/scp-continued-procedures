# How to suggest an idea...
## About adding an SCP to the game?
SCP object should meet this criteria:
1. SCP should not contain NSFW elements.
2. SCP should be not so popular (example - SCP-096).
   - Popular SCP can be added if:
      1. It is Christmas.
      2. If highly requested.

**Also p.3 does not apply on game's Author**
